See next page for how to define \AgdaFunction{two} (doesn't turn into a
link because the target hasn't been defined yet). We could do it
manually though; \hyperlink{two}{\AgdaDatatype{two}}.

\AgdaTarget{two}
\hypertarget{two}{}
\begin{code}%
\>[0]\AgdaFunction{two}\<[8I]%
\>[8I]\AgdaSymbol{:}\<[9I]%
\>[9I]\AgdaDatatype{ℕ}\<%
\\
\>[0]\AgdaFunction{two}\<[10I]%
\>[10I]\AgdaSymbol{=}\<[11I]%
\>[11I]\AgdaInductiveConstructor{suc}\<[12I]%
\>[12I]\AgdaSymbol{(}\AgdaInductiveConstructor{suc}\<[13I]%
\>[13I]\AgdaInductiveConstructor{zero}\AgdaSymbol{)}\<%
\end{code}

\AgdaInductiveConstructor{zero} is of type
\AgdaDatatype{ℕ}. \AgdaInductiveConstructor{suc} has not been defined to
be a target so it doesn't turn into a link.

\newpage

works automatically.

The AgdaTarget command takes a list as input, enabling several targets
to be specified as follows:

\AgdaTarget{if, then, else, if\_then\_else\_}
\begin{code}%
\>[0]\AgdaFunction{if\_then\_else\_}\<[21I]%
\>[21I]\AgdaSymbol{:}\<[22I]%
\>[22I]\AgdaSymbol{\{}\AgdaBound{A}\<[23I]%
\>[23I]\AgdaSymbol{:}\<[24I]%
\>[24I]\AgdaPrimitiveType{Set}\AgdaSymbol{\}}\<[25I]%
\>[25I]\AgdaSymbol{→}\<[26I]%
\>[26I]\AgdaDatatype{Bool}\<[27I]%
\>[27I]\AgdaSymbol{→}\<[28I]%
\>[28I]\AgdaBound{A}\<[29I]%
\>[29I]\AgdaSymbol{→}\<[30I]%
\>[30I]\AgdaBound{A}\<[31I]%
\>[31I]\AgdaSymbol{→}\<[32I]%
\>[32I]\AgdaBound{A}\<%
\\
\>[0]\AgdaFunction{if}\<[33I]%
\>[33I]\AgdaInductiveConstructor{true}\<[34I]%

\>[35I]\AgdaBound{t}\<[36I]%
